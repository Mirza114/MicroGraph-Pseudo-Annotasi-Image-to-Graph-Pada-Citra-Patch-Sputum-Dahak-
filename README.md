# MicroGraph

## Pengertian

MicroGraph adalah program Python untuk pseudo-anotasi citra patch sputum dahak dengan mengubah hasil segmentasi citra menjadi representasi graf.

## Cara Compile / Menjalankan

Install library:

```bash
pip install opencv-python numpy matplotlib
```

Jalankan program:

```bash
py src\graph_annotation_bacilli.py
```
